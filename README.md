# Treemap Visualization – Multi Dataset (Video Games, Movies & Kickstarter)

Questo progetto è stato sviluppato come parte del percorso **"Data Visualization"** di [freeCodeCamp](https://www.freecodecamp.org/).

L’obiettivo è realizzare una **visualizzazione interattiva a treemap** di diversi dataset (Videogiochi, Film e Kickstarter), utilizzando **D3.js (v7)** e rispettando le specifiche dei test ufficiali di FreeCodeCamp.  
L’interfaccia è stata realizzata in **HTML**, **CSS** e **JavaScript puro**, con un design pulito, accessibile e supporto al **tema chiaro/scuro** persistente.

---

### 🧠 Obiettivi

- Rappresentare graficamente i dati gerarchici (categorie e valori) tramite una **treemap interattiva**
- Supportare **più dataset** selezionabili dinamicamente (Videogiochi, Film, Kickstarter)
- Implementare un **tooltip dinamico** con informazioni contestuali
- Gestire **legenda e scala cromatica** coerente per ciascun dataset
- Gestire un **tema dark/light** sincronizzato con le preferenze di sistema e salvato in `localStorage`
- Integrare un **toggle tema chiaro/scuro** sincronizzato con le preferenze di sistema
- Garantire una **presentazione responsive e accessibile** su diversi dispositivi

---

### ⚙️ Funzionalità principali

- Visualizzazione **treemap interattiva** con D3.js
- **Cambio dataset** istantaneo tramite pulsanti (Games / Movies / Kickstarter)
- Tooltip contestuale con nome, categoria e valore
- Layout completamente **responsive** e ottimizzato per mobile
- Effetti di **hover, blur e shadow** coerenti con il tema attivo
- Tema **chiaro/scuro persistente**, commutabile via icona 🌙/☀️
